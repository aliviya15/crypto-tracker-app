#  Crypto & Stock Tracker Web App

## 🔰 Project Overview

The Crypto & Stock Tracker Web App is a user-friendly web application that allows users to search for cryptocurrencies or stocks and view their real-time market data. This project focuses on integrating public APIs, building interactive UI components, and implementing modern JavaScript concepts.

---

##  Objective

The main objective of this project is to:

* Demonstrate API integration using JavaScript (`fetch`)
* Build an interactive and responsive user interface
* Implement core functionalities like search, filtering, and sorting

---

##  API Used

Example API Endpoint:
https://api.coingecko.com/api/v3/coins/{coin_name}

---

## ✨ Features (Planned & Implemented)

* 🔍 Search for cryptocurrencies (e.g., Bitcoin, Ethereum)
* 📊 Display real-time price and market data
* 📈 Show price change (24h)
* 🎯 Filter coins based on criteria
* 🔃 Sort data (ascending / descending)
* ❤️ Add to favorites (planned)
* 🌙 Dark Mode / Light Mode (planned)



## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript 
* Fetch API



## ⚙️ How It Works

1. User enters a cryptocurrency name in the search bar
2. The app sends a request to the CoinGecko API
3. The API returns real-time data
4. The app dynamically displays the data on the screen

##  How to Run the Project

1. Clone or download the repository
2. Open the folder
3. Open `index.html` in your browser
4. Enter a coin name (e.g., bitcoin)
